System Architecture

This API bridges the gap between massive S3 Storage and the need for Real-time Search.

  • Backend: Go (Gin Gonic) with singleflight for thundering-herd protection.
  • Database: PostgreSQL with pg_trgm for fuzzy matching.
  • CDN: CloudFront integration for globally distributed asset delivery.
  • Sync: Real-time S3 metadata updates via AWS SQS.

API Endpoints

GET /explorer Web interface for file browsing
GET /file/*filepath Direct access with smart redirect & cache
GET /search?q={query} Fuzzy search for S3 metadata
GET /health System health & DB connectivity
